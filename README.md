# 📊 Interactive Sales Dashboard in Microsoft Excel

An interactive Sales Dashboard built entirely in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, and VBA Macros.

The dashboard allows users to dynamically analyze sales performance across different regions with a single click.

# Project Overview

This project demonstrates how Excel can be used as a powerful Business Intelligence tool.

The dashboard provides interactive insights into:

- Total Sales
- Target Hit %
- Away From Target %
- Sales Executive Performance
- Region-wise Analysis

Users can filter the complete dashboard using region buttons powered by VBA macros.

# How to Use

1. Download the repository.
2. Open **Sales_Dashboard.xlsm**.
3. Click **Enable Editing**.
4. Click **Enable Content (Macros)**.
5. Use the Region buttons or slicers to filter data.
6. Explore the dashboard interactively.
